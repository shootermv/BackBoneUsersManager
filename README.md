# UsersManager — the BackboneJS app
Simple  experimental backbone.JS application for managing user accounts in the site BackOffice.
The frameworks used:

* 1.Microsoft MVC3 - as server
* 2.twitter bootstrap -as html/css
* 3.backbonejs -as javascript
 
### Running the app during development

You can run this application with WebMatrix (Open Site->Directory as site) or Vs Express (free tools)
Or VisualStudio (payment)
